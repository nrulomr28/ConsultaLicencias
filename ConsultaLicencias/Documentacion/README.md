# Consulta Licencias

## Descripción

Consulta Licencias es un sistema legado utilizado para la consulta de información histórica relacionada con licencias y sus trámites asociados.

El sistema antecede funcionalmente a SIGELIC y se emplea principalmente como fuente de consulta de expedientes y documentación histórica.

Actualmente se encuentra en proceso de análisis, documentación y preservación.

---

## Objetivos del rescate

* Preservar el código fuente original.
* Identificar dependencias y configuración requerida.
* Documentar el modelo de datos utilizado.
* Analizar los mecanismos de consulta de documentación histórica.
* Determinar la viabilidad de exponer la información mediante servicios web o integraciones futuras.
